# بيّن (Bayyin) — محاكاة المحاكمات القضائية السعودية بوكلاء متعددين

نظام يحاكي إجراءات التقاضي في القضاء السعودي (المرحلة الابتدائية ثم الاستئناف
والتماس إعادة النظر) ليتدرّب المحامون على قضاياهم في بيئة آمنة.

## المبدأ الحاكم
> **الإجراء القضائي = كود حتمي. المحتوى القانوني = نموذج لغوي مُسنَد بمصدر.**

تسلسل الإجراء والمواعيد وقابلية الاعتراض تُحسم بقواعد بكود قابلة للتدقيق
(`rules.py`)، بينما يولّد النموذج المحتوى (المذكرات، التسبيب، المنطوق) مُلزَماً
بالاستشهاد من قواعد `file_search`، وتتحقّق منه بوابة الاستشهاد (`citations.py`).

## المعمارية (M1 — شريحة رأسية للمرحلة الأولى)
- **التنسيق:** LangGraph StateGraph — كل مرحلة عقدة، وكل انتقال محفوظ بـ checkpoint.
- **النماذج:** `gpt-5.4-mini` (موجِّه) · `gpt-5.4` (وكلاء) · `gpt-5.4-pro` (حكم/شائك).
- **الإسناد (RAG):** 4 قواعد متجهية مستضافة عبر `file_search` + بحث ويب مقيّد بنطاقات رسمية.

التدفّق: `router → intake_register → notify_defendant → defendant_response →
plaintiff_reply (تدخّل بشري) → defendant_rejoinder → close_pleadings → judgment →
serve_judgment`.

## التشغيل
```powershell
pip install -r requirements.txt

# عرض كامل من القيد حتى الحكم (وضع وهمي، بلا API):
python run_demo.py

# الاختبارات الحتمية (بلا API):
python -m pytest -q

# الوضع الحقيقي (يتطلب OPENAI_API_KEY في .env):
$env:BAYYIN_MOCK=0 ; python run_demo.py
```

## بنية الملفات
| الملف | الدور |
|---|---|
| `settings.py` | النماذج، معرّفات القواعد المتجهية، النطاقات، الحدود النظامية |
| `state.py` | `CaseState` والكائنات القانونية (Document/Ruling/Citation) |
| `rules.py` | قواعد الإجراء الحتمية (تدقيق، قابلية اعتراض، أسباب المادة 200) |
| `citations.py` | بوابة الاستشهاد ضد الهلوسة القانونية |
| `llm.py` | غلاف Responses API + نموذج وهمي |
| `tools.py` | بُناة أدوات `file_search` / `web_search` |
| `prompts.py` | المطالبات النظامية للأدوار (عربي) |
| `nodes.py` | سلوك كل دور (عُقَد LangGraph) |
| `graph.py` | آلة الحالة الإجرائية |

## تنبيهات دقّة
- **حدّ القطعية 50,000 ر.س غير مؤصَّل نظامياً** (`SUMMARY_COURT_FINALITY_VERIFIED=False`)؛
  يجب تأصيله من `boe.gov.sa` قبل الإنتاج.
- `uqn.gov.sa` = جريدة أم القرى الرسمية (لا منصة المنافسات).
- المخرجات **محاكاة تدريبية** لا أحكام حقيقية ولا استشارة قانونية.

## التالي (خارطة الطريق)
- **M2:** حلقة تبادل المذكرات الحقيقية + وكيل الخبير + منطق التشبّع.
- **M3:** الاستئناف + دائرة الثلاثة قضاة (نماذج متنوعة، جولة عمياء) ضد التحيّز.
- **M4:** التماس المادة 200 + وضع حقن الثغرات الإجرائية (LegalSim).
- **M5:** تأصيل الإسناد بالاسترجاع الفعلي + تقييم بـ ALARB + Postgres checkpointer.
